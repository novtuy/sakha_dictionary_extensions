Privacy Policy

This extension does not collect, store, or transmit any personal data.

All data created by the user (flashcards, words, learning progress) is stored locally in the browser using IndexedDB and is never sent to third parties.

The extension performs network requests only to https://igi.ysn.ru/ to fetch dictionary or reference data explicitly requested by the user. No user-identifying information is included in these requests.

The extension does not track browsing activity and does not perform background monitoring.
